# 기간제근로자 출근관리부 웹앱

GitHub Pages에 올릴 수 있는 정적 웹앱입니다. 브라우저에서 `assets/기간제출근부.xlsx` 템플릿을 읽고 입력값을 채운 뒤 새 엑셀 파일을 다운로드합니다.

## 사용 방법

1. `index.html`을 브라우저에서 엽니다.
2. 작성도서관, 근로자, 근무년월, 근무시간, 작성자, 확인자 정보를 입력합니다.
3. 근무요일과 휴일을 설정합니다.
4. 화면 아래의 `문서 미리보기`에서 반영 내용을 확인합니다.
5. `엑셀 파일 생성`을 누릅니다.

## GitHub Pages 배포

1. 이 `timeSheetWeb` 폴더 내용을 GitHub 저장소에 올립니다.
2. 저장소 `Settings` → `Pages`에서 배포 브랜치와 폴더를 선택합니다.
3. 배포된 주소에서 `index.html`이 열리면 바로 사용할 수 있습니다.

## 셀 매핑

- `A3`: 근무기간
- `C4`: 작성도서관 이름
- `O4`: 근로자 이름
- `I5`: 평일/주말 표시
- 날짜 영역: 기존 Python 프로그램과 같은 C~Q 날짜 블록
- `H39`: 작성자 소속 문구
- `N39`: 작성자 이름
- `H41`: 확인자 도서관 문구
- `N41`: 확인자 이름

## 미리보기

입력값, 근무요일, 휴일 목록이 바뀌면 문서 미리보기가 즉시 다시 그려집니다. 실제 다운로드되는 엑셀 파일은 같은 날짜 계산과 휴일 처리 로직으로 생성됩니다.

## 참고

엑셀 생성은 브라우저용 ExcelJS CDN을 사용합니다. GitHub Pages처럼 인터넷이 되는 환경에서 동작합니다. `template-data.js`에는 같은 엑셀 템플릿이 base64로 포함되어 있어 로컬에서 `index.html`을 직접 열어도 템플릿을 읽을 수 있습니다.
